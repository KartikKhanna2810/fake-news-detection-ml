# Fake News Detection using Machine Learning

# Overview

This project focuses on detecting fake news articles using Natural Language Processing (NLP) and Machine Learning techniques. The model is trained on a real-world dataset containing thousands of labeled news articles.

# Features

* End-to-end NLP pipeline
* Text preprocessing using regex and cleaning techniques
* TF-IDF vectorization for feature extraction
* Multiple ML models implemented and compared
* High accuracy classification (best model: Linear SVM)

# Tech Stack

* Python
* Scikit-learn
* Pandas, NumPy
* TF-IDF (Text Vectorization)
* NLP techniques

# Models Used

* Logistic Regression
* Random Forest
* Naive Bayes
* Support Vector Machine (Best Performing)

# Dataset

* Fake.csv
* True.csv
  (Sourced from Kaggle Fake News Dataset)

# Results

* Achieved high accuracy (~99%) using Linear SVM
* Evaluated using accuracy, precision, recall, and F1-score

# How to Run

1. Open the notebook in Google Colab or Jupyter Notebook
2. Upload dataset files (Fake.csv & True.csv)
3. Run all cells sequentially

# Project Structure

* `Project.ipynb` → Main notebook
* `Fake.csv`, `True.csv` → Dataset
* PPT & Report files for documentation

# Conclusion

The project demonstrates how classical ML models can effectively detect fake news using textual data and feature engineering.
